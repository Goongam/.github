# 프로젝트 OpenBook

### 📙 개요
<p align="center"><img src="https://github.com/Project-OpenBook/.github/assets/126096318/caf12569-f0ec-420e-acd8-7c9b3052cf3a" width="200" height="200"/></p>
'OpenBook' 은 일반인들도 손쉽게 행사와 부스를 운영하고 관리할 수 있도록 돕는 웹 플랫폼입니다. 이 플랫폼은 행사 및 부스의 실시간 정보 전달, 
상품 관리, 사용자 후기 및 피드백 수집, 상세 정보 제공 등을 통해 행사 참여자와 운영자 모두에게 최적의 사용자 경험을 제공합니다. OpenBook'은 사용자들이 행사와 부스 정보를 쉽게 접근하고, 원하는 정보를 효율적으로 얻을 수 있도록 설계되었습니다.
